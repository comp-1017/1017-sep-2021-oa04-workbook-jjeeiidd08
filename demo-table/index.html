Table Demo

A table is a way of creating a structured grid display of data. The tags we will be interested are:

  • <table> : identifies the table
  • <thead> identifies the header row of of the table
  • <th> : identifies the text in the head of the column
  • <tbody> : identifies the body of the table (detail rows)
  • <tr> : identifies the start of a body row
  • <td> : identifies a data cell of the body row
  • <tfoot> : identifies the footer row of the table

All tags are paired tags (opening and closing tag). The border parameter on the table tag will allow you to specify a border around your table. The colspan tag and rowspan tag used on the <td> tab allows you to merger columns on a single row and columns cell across rows.
